# Develop 

- Environment: 推荐使用Python虚拟环境，以此来屏蔽Python版本的差异
- Decpendency: 所有需要的依赖都会被记录到`requirement.txt`中, 使用`pipreqs /path/to/learn-python`获取最小化依赖列表
- Format: 强制使用`black`做代码格式化，统一代码样式
- CI Check: 先在本地做CI模拟检测, 避免CI流水的长时间占用
